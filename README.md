## 📁 Files

```tree
└─01_week
    ├─part01
    └─part03
```

-   N주차에 N_week 디렉토리를 생성합니다.
-   해당 디렉토리의 README.md를 작성합니다. (예시 파일은 1주차 디렉토리 참고)
-   발표 자료 파일은 자유롭게 저장하되 디렉토리는 따로 관리합니다.
-   발표 후 README.md 및 N_week/README.md에 목차를 정리합니다.
